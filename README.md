# GuruCool - Smart School Management System

GuruCool is a comprehensive school management platform built for Indian schools. It streamlines attendance automation, communication, academic management, and administrative operations.

## Features

- **Attendance Automation** - Photo-based attendance with WhatsApp notifications
- **Communication Hub** - Direct WhatsApp integration for parent-teacher communication
- **Academic Management** - Grades, assignments, and progress tracking
- **Multi-School Support** - Manage multiple branches from one dashboard
- **Mobile Access** - Full functionality on any device

## Technology Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
